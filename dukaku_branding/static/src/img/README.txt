Place the following files in this directory:

  logo.png     — Login page logo. Recommended: 300 × 100 px, transparent background, PNG.
  favicon.ico  — Browser tab icon. Standard: 16×16 and 32×32 multi-size ICO, or a 32×32 PNG renamed .ico.

The backend navbar logo comes from:
  Settings > Companies > (your company) > upload logo image

That logo is stored in the database (res.company.logo), not as a static file.
